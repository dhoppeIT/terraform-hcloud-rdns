# terraform-hcloud-rdns

This example will manage the Hetzner Cloud DNS and creates a single reverse DNS entry.

## Graph

<img alt="Graph" src="https://github.com/dhoppeIT/terraform-hcloud-rdns/blob/main/examples/single_entry/rover.svg?raw=true" width="100%" height="100%">

## Usage

To run this example, you need to execute the following commands:

```shell
$ terraform init
$ terraform plan
$ terraform apply
```

:warning: This example may create resources that cost money. Execute the command
`terraform destroy` when the resources are no longer needed.
